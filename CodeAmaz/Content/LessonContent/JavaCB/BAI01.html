Bài tập

Bạn hãy thay dấu ... bằng các đoạn code thích hợp để chương trình hiển thị ra màn hình dòng chữ:

Hello, World!

Hãy đọc phần lý thuyết và xem hướng dẫn để biết cách làm bài này!

Lý thuyết

Hiển thị ra màn hình dòng chữ Hello, World! là một trong những chương trình cơ bản nhất khi bắt đầu làm quen với một ngôn ngữ lập trình. Trong Java để hiển thị ra màn hình một dòng chữ nào đó bạn có thể sử dụng câu lệnh System.out.println(). Ví dụ để hiển thị ra màn hình dòng chữ "Hello Java" bạn có thể làm như sau:

public class HelloWorld {

        public static void main(String[]args) {

               System.out.println("Hello Java");

        }

}

Kết quả khi chạy chương trình:

Hello Java

Nếu bạn là người mới bắt đầu thì trong đoạn code trên có khá nhiều câu lệnh mà bạn không biết như public class HelloWorld và public static void main(String[]args) nhưng bạn đừng quan tâm tới các câu lệnh này vội, hãy coi như những câu lệnh này là bắt buộc đối với 1 chương trình Java.

Tương tự như trên, bạn có thể viết chương trình hiển thị ra màn hình tên của mình. Ví dụ nếu bạn tên là Nguyen Van Nam thì bạn có thể làm như sau:

public class HelloWorld {

        public static void main(String[]args) {

               System.out.println("Nguyen Van Nam");

        }

}

Đọc tới đây bạn đã biết cách sử dụng câu lệnh System.out.println() để hiển thị thông tin ra màn hình, hãy quay lại phần bài tập và làm thử.

Nếu bạn vẫn chưa quen cách sử dụng các câu lệnh trong Java thì có thể xem hướng dẫn giải bài này ở bên dưới.

Hướng dẫn

Để làm bài này bạn chỉ cần điền System.out.println("Hello, World!"); vào dấu ...

Code mẫu:

public class HelloWorld {

        public static void main(String[]args) {

               System.out.println("Hello, World!");

        }

}
